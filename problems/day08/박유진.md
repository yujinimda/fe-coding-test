## 에러 바운더리와 Suspense를 활용한 사용자 정보 로딩

**카테고리:** ⚛️ React
**난이도:** 🟡 Medium
**제한 시간:** 15분

---

### 문제

사용자 정보를 비동기로 불러오는 컴포넌트를 구현하세요. 이 컴포넌트는 에러가 발생할 경우 에러 메시지를 보여주고, 로딩 중일 때는 로딩 메시지를 보여줘야 합니다. 에러 바운더리와 Suspense를 활용하여 구현하세요.

### 코드

```typescript
import React, { Component, Suspense } from 'react';

/**
 * 사용자 데이터를 비동기로 가져오는 함수
 * @returns {Promise<{name: string, age: number}>} 
 * @throws 50% 확률로 에러 발생 
 */
function fetchUserData() {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      if (Math.random() > 0.5) {
        resolve({ name: 'John Doe', age: 30 });
      } else {
        reject('Failed to fetch user data');
      }
    }, 1000);
  });
}

/**
 * React.lazy를 활용한 비동기 사용자 데이터 컴포넌트
 *
 * React.lazy는 { default: Component } 형태의 Promise를 기대함 (안그러면 오류)
 * 원래 동적 import용으로 만들어졌기 때문이다!
 * import()는 ES 모듈을 가져오는데 export default로 내보낸 컴포넌트는 { default: 컴포넌트 } 형태로 들어옴
 * 그래서 React.lazy는 항상 { default: 컴포넌트 } 구조를 기대
 * 
 * fetchUserData()의 결과를 컴포넌트로 감싸서 반환하고
 * 에러 발생 시 Error를 throw하여 ErrorBoundary에서 잡히도록 함
 */
const UserData = React.lazy(() => {
  return fetchUserData()
    .then((user) => ({
     default: () => <p>User: {user.name}, Age: {user.age}</p>
    }))
    .catch((err) => {
      throw new Error(err);
    });
});

/**
 * 에러 바운더리 클래스 컴포넌트
 *
 * 하위 컴포넌트에서 발생한 렌더링 에러를 잡아서
 * 앱 전체가 깨지는 대신 에러 메시지를 표시함
 *
 * @extends {Component<{children: React.ReactNode}, {hasError: boolean}>}
 */
class ErrorBoundary extends Component {
  constructor(props) { // ErrorBoundary가 만들어질 때 자동 실행(함수 컴포넌트로 비유하면 useState의 초기값 설정이랑 같은 역할)
    super(props);  // 부모(Component)야 props 받아
    this.state = { hasError: false }; // 나(ErrorBoundary)의 state를 초기화
  }

   static getDerivedStateFromError() {
    return { hasError: true };
  }

  render() {
    if (this.state.hasError) { // 나의 state에서 hasError 확인
      return <p>Something went wrong while loading user data.</p>;
    }
    return this.props.children; // 나한테 전달된 자식 컴포넌트 렌더링
  }
}

/**
 * 앱 루트 컴포넌트
 *
 * 구조: ErrorBoundary → Suspense → UserData
 * - ErrorBoundary: 에러 발생 시 에러 메시지 표시
 * - Suspense: 비동기 로딩 중 로딩 메시지 표시
 * - UserData: 사용자 데이터 렌더링
 */
function App() {
  return (
    <ErrorBoundary>
      <Suspense fallback={<p>Loading user data...</p>}>
        <UserData />
      </Suspense>
    </ErrorBoundary>
  );
}
export default App;
```

### 요구사항

1. 에러가 발생하면 'Something went wrong while loading user data.' 메시지를 보여줘야 합니다.
2. 로딩 중일 때는 'Loading user data...' 메시지를 보여줘야 합니다.
3. 사용자 데이터가 성공적으로 로드되면 'User: [name], Age: [age]' 형식으로 보여줘야 합니다.
